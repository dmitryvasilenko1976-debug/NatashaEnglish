## 📚 ELT-БЛОК — Педагогические фичи (реализовано 2026-06-24)

> Анализ с позиции специалиста по методике преподавания EFL/EAP.
> Приоритет — продуктивные навыки и специфика медицинского английского.

### Что реализовано

- [x] **Морфология** (`MorphologyScreen` + `MorphologyQuizScreen`)
  - 48 медицинских морфем: 16 приставок, 16 корней, 16 суффиксов
  - Accordion с примерами и подсказками → тест (морфема → значение, +5 XP)
  - Принцип Nation (2001): vocabulary через морфемный анализ = понимание тысяч терминов

- [x] **Ложные друзья** (`FalseFriendsScreen`)
  - 15 критических пар (angina/ангина, decade/декада, physician/физик, ...)
  - Встроенный квиз прямо в экране списка, красные КРИТИЧЕСКИ-блоки
  - Специфика для российских врачей — ошибки с реальными клиническими последствиями

- [x] **Клоуз-тест** (`ClozeScreen`)
  - Заполни пропуск на основе сохранённых слов в контексте реальных статей
  - Принцип Krashen: понимание → использование (i+1 scaffolding)
  - Пустое состояние с мотивирующей подсказкой: «Сохрани слова во время чтения»

- [x] **Озвучка предложений** (TTS в `ReadingScreen`)
  - Web Speech API: кнопка 🔊 в нижней навигации статьи
  - Скорость 0.82× для чёткости (оптимально для медицинских терминов)
  - Только web (Platform.OS === 'web'), не мешает мобильному

- [x] **Режим вспоминания** (`QuizScreen` + `QuizSelectScreen`)
  - Продуктивный recall: показ перевода (RU) → угадать английское слово
  - Два режима в QuizSelectScreen: «Квиз →» (узнавание) и «← Вспомни» (вспоминание)
  - По теории дискретного компонента (Nation): узнавание ≠ вспоминание

### ELT-принципы, заложенные в основу

| Принцип | Реализация |
|---|---|
| Nation's vocabulary learning | морфология + клоуз + spaced repetition |
| Focus on Form | FalseFriends: форма vs значение |
| Receptive → Productive | Квиз (рецептивный) → Вспоминание (продуктивный) |
| Authentic context | Клоуз из реальных медицинских статей |
| TTS pronunciation | Web Speech API в ReadingScreen |

## ◈ ЭПОХА III — "ЛИГА И ИСПЫТАНИЕ"

**Психологический крючок:** Social Comparison Theory. Люди работают усерднее когда соперники близки по уровню. Ключ: 2–3 соперника всегда немного впереди, остальные позади.

### 3.1 Симулированная Лига
**Статус:** `[ ]` не начато

**Алгоритм:**
- 2 соперника держатся на `user.xp + (50..150)` — давление

**Уровни лиги:** Лига Хоббита → Лига Странников → Лига Рейнджеров → Лига Мудрецов

## ◈ ЭПОХА IV — "СВИТОК ПАМЯТИ"
### Слова, возвращающиеся в нужный час

**Психологический крючок:** Spaced Repetition (SM-2). Самый научно обоснованный метод долгосрочного запоминания. Слово показывается ровно когда вот-вот забудется.

---

### 4.1 Алгоритм SM-2
**Статус:** `[ ]` не начато

**Структура данных для каждого слова:**
```json
{
  "dyspnea": {
    "interval": 1,
    "easeFactor": 2.5,
    "nextReview": "2026-06-24",
    "repetitions": 0
  }
}
```

**Формула:**
- Оценка Again/Hard (0–2): interval = 1, ef -= 0.15
- Оценка Good (3–4): interval = prev × ef
- Оценка Easy (5): interval = prev × ef × 1.3, ef += 0.1

**Файлы:**
```
gamificationService.js  — sm2Update(wordData, grade) → обновлённый wordData
storageService.js       — getWordsForReview() — слова где nextReview <= today
                        — updateWordSRS(articleId, word, grade)
```
